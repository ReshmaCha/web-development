Maak een eigen voorbeeld van de volgende functies en licht toe wat de functie doet in het blok:

1. charAt ( )

staat voor karakter na (character at) Tekens in een string worden van links naar rechts geïndexeerd.

De index van het eerste teken is 0, en de index van het laatste teken - in een string genaamd stringName- is stringName.length - 1.

Als het indexopgegeven item buiten dit bereik valt, retourneert JavaScript een lege tekenreeks.

Voorbeeld in script en het antwoord wat hier uitkomt is The character at index 0 is 'B'


2.concat( )

staat voor: de stringargumenten samen met de aanroepende string en retourneert een nieuwe string.

Voorbeeld : de uitkomst hiervan is Hello World en daaronder voor de volgende console ,World Hello


3. slice( )

betekent dat er delen uit worden gehaald/getrokken.

De methode slice () extraheert delen van een string en retourneert de geëxtraheerde delen in een nieuwe string.

Gebruik de begin- en eindparameters om het deel van de tekenreeks op te geven dat u wilt extraheren.

Het eerste teken heeft positie 0, het tweede heeft positie 1, enzovoort.

Tip: gebruik een negatief getal om te selecteren vanaf het einde van de string.

De slice()methode extraheert een sectie van een string en retourneert deze als een nieuwe string, zonder de originele string te wijzigen.

In het voorbeeld komt dan : the lazy dog , als uitkomst.


4. split( )

De splitmethode verdeelt een String in een geordende lijst van substrings,

plaatst deze substrings in een array en retourneert de array.

De deling wordt gedaan door te zoeken naar een patroon; waarbij het patroon wordt opgegeven als de eerste parameter in de aanroep van de methode.


5. substr( )

De substr() methode retourneert een deel van de tekenreeks, beginnend bij de opgegeven index en daarna een bepaald aantal tekens.
