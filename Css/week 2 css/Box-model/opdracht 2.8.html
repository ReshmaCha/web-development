Box-Sizing

Standaard wordt de breedte en hoogte van een element als volgt berekend:

breedte + opvulling + rand = werkelijke breedte van een elementhoogte + opvulling + rand = werkelijke hoogte van een element.

Dit betekent: Wanneer u de breedte/hoogte van een element instelt, wordt het element vaak groter weergegeven dan u hebt ingesteld (omdat de rand en opvulling van het element worden toegevoegd aan de opgegeven breedte/hoogte van het element).

Met box-sizing los je het probleem op !
