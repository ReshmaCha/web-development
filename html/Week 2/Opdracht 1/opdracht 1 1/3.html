1. Je kunt CSS op 3 manieren in je pagina stoppen. Je kunt het in een apart bestand stoppen dat eindigd op .css. We spreken dan over een stylesheet. Je kunt het ook in een speciaal gedeelte in de header zetten. Als laatste kun je het ook nog direct in de tag zetten.

2. Een syntax is de manier waarop iets geschreven is , een syntax in CSS verwijst naar datgene wat gestylt wordt.

Een declaratie-block bestaat uit minstens een declaratie eindigend op puntkomma's

Een declaratie bestaat altijd uit een property(eigenschap,attribuut) en een value(waarde) gescheiden door een dubbele punt.

Meerdere declaraties worden gescheiden door puntkomma's

Een declaratie wordt omringd met brackets (krullende beugels)
