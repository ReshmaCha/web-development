Wat zou eventueel de voor en nadelen t.o.v Object constructors?

Een objectinitializer is een uitdrukking waarmee we een nieuw gemaakt object kunnen initialiseren.

Het is een door komma's gescheiden lijst van nul of meer paren van eigenschapsnamen en bijbehorende waarden van een object tussen een paar accolades {} Zoals ik het begrijp is het handig om een iniatilezer te gebruiken zodat je nieuwe info makkelijk kan toevoegen aan de bestaande info.( In een constructors?) https://developer.mozilla.org/nl/docs/Web/JavaScript/Guide/Werken_met_objecten
