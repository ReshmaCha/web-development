Transitions

Met transities wordt bedoeld een overgang van een begintpunt tot een eindpunt

Je kan de snelheid ermee aanpassen hoelang het duurt voor het van punt 1 naar punt 2 gaat

Je kan hiermee mooie animaties maken zonder Javascript te hoeven te gebruiken

Geef altijd de eigenschap overgangsduur op, anders is de duur 0 en heeft de overgang geen effect.
