Wat zijn rest parameters? Geef een voorbeeld.

De rest parameter syntax laat ons toe om een onbepaald aantal argumenten te vertegenwoordigen als een array.

Verschillen tussen de rest parameter en het arguments object

Er zijn drie belangrijke verschillen tussen de rest parameters en het arguments object:

rest parameters zijn alleen de argumenten die niet via een andre naam aan de functie zijn doorgegeven,

terwijl het arguments object wel alle argumenten bevat die zijn doorgegeven.

het arguments object is niet echt een array, terwijl de rest parameters wel een instantie van een Array zijn,

waardoor je methodes zoals sort, map, forEach en pop rechtstreeks kan gebruiken.

de argumenten object heeft extra functionaliteit specifiek voor zichzelf (zoals de callee property).
